# 🐟 Su Ürünleri AR

Su Ürünleri AR, artırılmış gerçeklik (AR), üç boyutlu modelleme ve yapay zekâ teknolojilerini kullanarak kullanıcıların su ürünlerini etkileşimli şekilde öğrenmesini sağlayan web tabanlı bir eğitim uygulamasıdır.

Kullanıcılar balık görsellerini kamera yardımıyla tarayabilir, ilgili balık türleri hakkında bilgi edinebilir, üç boyutlu modelleri inceleyebilir ve quiz sistemi ile bilgilerini test edebilirler.

---

# 🎯 Projenin Amacı

Bu proje, geleneksel öğrenme yöntemlerini artırılmış gerçeklik teknolojileri ile destekleyerek su ürünleri eğitiminin daha etkili ve eğlenceli hale getirilmesini amaçlamaktadır.

Proje kapsamında kullanıcılar:

- Balık türlerini tanıyabilir
- Balıklar hakkında bilgi alabilir
- 3D modelleri inceleyebilir
- Yapay zekâ ile balık analizi yapabilir
- Quiz sistemi ile öğrendiklerini test edebilir

---

# 🚀 Özellikler

### 📷 Artırılmış Gerçeklik (AR)

- Kamera üzerinden hedef görsel tanıma
- Gerçek zamanlı AR deneyimi
- Web tabanlı kullanım

### 🐠 Balık Bilgilendirme Sistemi

- Balık adı
- Habitat bilgisi
- Beslenme şekli
- Ortalama boy
- Koruma durumu

### 🎮 Quiz Sistemi

- Çoktan seçmeli sorular
- Puan hesaplama
- Sonuç ekranı

### 🧠 Yapay Zekâ Destekli Analiz

- Fotoğraf yükleme
- Balık türü tahmini
- Tür açıklaması üretimi

### 🎨 3D Model Görüntüleme

- GLB formatında modeller
- Döndürme ve yakınlaştırma desteği
